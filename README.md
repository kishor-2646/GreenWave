## 🌟 Overview

🚑 **GreenWave** is a real-time smart traffic management system designed to create a **green corridor for ambulances**.

It tracks ambulance movement and **automatically controls traffic signals** to ensure **faster and safer emergency transport**.

---

## 🎯 Problem Statement

- 🚧 Heavy traffic delays ambulances
- ⏱️ Delay in reaching hospitals costs lives
- 🚦 Manual traffic control is inefficient..

---

## 💡 Solution

GreenWave solves this by:

✔️ Real-time ambulance tracking  
✔️ Automated traffic signal control  
✔️ Alerts to traffic police for non-automated junctions  
✔️ Smart route optimization  .

---

## 🧠 Key Features

🚑 Live Ambulance Tracking  
📍 GPS-based location updates  
🚦 Smart Traffic Signal Control  
👮 Police Alert System  
📊 Admin Dashboard

## 🚀 Getting Started

### 1️⃣ Clone the repository
```bash
git clone https://github.com/kishor-2646/GreenWave.git
cd GreenWave
```

### 2️⃣ Install dependencies
```bash
flutter pub get
```
### 3️⃣ Run the app
```bash
flutter run
```

### 🔐 Environment Setup

**Create a .env file and add:**
```bash
GOOGLE_MAPS_API_KEY=your_api_key
FCM_SERVER_KEY=your_key
```
